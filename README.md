## AWS on Windows 10

AWS cli on windows 10 does not work well with `git-bash`
for that reason we need to add a ~/.bashrc file
with the path to the AWS Cli exe on windows:

my .bashrc:  `alias aws='C:/Program\ Files/Amazon/AWSCLI/aws.exe'`
